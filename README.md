# All 4 All

This application provides a platform where organizations and volunteers can post and browse for events in their local area. Volunteers are rewarded with badges for completing events. All 4 All provides a one stop shop where users can create / edit their profiles, register for events, and find new opportunities. This app aims to spread awareness and make the volunteer process easier for everyone.

# React + Vite

To run the frontend:
cd src
npm run dev

To run the backend:
open docker desktop, then: "docker compose up"
run "node src/backend/server.js" in a separate terminal
